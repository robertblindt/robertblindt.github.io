Keyboard Questionnaire

I used to have a lot of bad tendencies when I typed, and they exacerbated some of the fine motor skill related issues I was having. I started using 'ergonomic keyboards' that aimed to straighten out the typing action but most of the ones that I could find were imperfect and very bulky, and my desk is not very deep. I came across a few commercially available split ortholinear keyboards, but they were very expensive so I built my own and create a custom keymap!

In short, there is a reprogrammable chip inside each of the sides that I can flash to change the keyboard layouts. Unlike most normal keyboards, there are action keys that you can use to move between 'keyboard layers' to overcome the reduced number of keys (kind of like numlock, but it totally changes the destination key instead of the modifier).

In my keyboard I have 4 layers. A primary layer (Layer 0), 2 layers that are controlled by 'momentary switches' (Layer 1 and Layer 2), meaning the layer becomes active while the button is being pressed, and a toggled layer (Layer 3), meaning I just need to click the button to move into and out of the mode. Layer 3 is specifically for video games, however the momentary switched layers I am constantly using for productivity. There are some mapped buttons that I do not regularly use, like the windows shortcuts and RGB controls, and also some layers with multiples of the same key like the square brackets.

The goal at the end of the day is to limit the amount of distance that each finger has to move for any given key, but keep functionality. My question for you is:

Things like Shift + Ctrl + Right are achievable by sequencing Shift > [MO1 + Right] + Ctrl )
